About
 
CONMET (CONstruct Measurement Evaluation Tool) is written by Leander De Schutter, a PhD Candidate at the Rotterdam School of Management. Feel free to send an email to deschutter@rsm.nl in case you find any bugs, have any questions about the app, or want to connect.

Currently, you can cite ConMET by referring to the R package:
De Schutter, L. (2021). conmet: Construct Measurement Evaluation Tool. R package version 0.1.0. https://CRAN.R-project.org/package=conmet

The application relies heavily on the lavaan package (Rosseel, 2012) and uses functions from the semTools package (Jorgensen et al., 2020) to obtain reliabilities. Please make sure to cite these packages when using the output of the application.

References
Jorgensen, T. D., Pornprasertmanit, S., Schoemann, A. M., & Rosseel, Y. (2020). semTools: Useful tools for structural equation modeling. R package version 0.5-3. Retrieved from https://CRAN.R-project.org/package=semTools
Rosseel, Y. (2012). lavaan: An R Package for Structural Equation Modeling. Journal of Statistical Software, 48(2), 1-36. URL http://www.jstatsoft.org/v48/i02/.

About
 
CONMET (CONstruct Measurement Evaluation Tool) is written by Leander De Schutter, a PhD Candidate at the Rotterdam School of Management. Feel free to send an email to at deschutter@rsm.nl in case you find any bugs.

The application relies heavily on the lavaan package (Rosseel, 2012) and uses functions from the semTools package (Jorgensen et al., 2020) to obtain reliabilities. Please make sure to cite these packages when using the output of the application.

Other packages that the app uses are:
  • foreign (R Core Team, 2020) to read datasets
  • shiny (Chang et al., 2021), shinydashboard (Chang & Ribeiro, 2018), waiter (Coene, 2021), DT (Xie, Cheng, & Tan, 2020) and shinyWidgets (Perrier, Meyer & Granjon, 2021) for the web application framework and features.
  • purrr (Henry & Wickham, 2020), stringr (Wickham, 2018), dplyr (Wickham, François, Henry, & Müller, 2020) for data and string wrangling.
  • summarytools (Comtois, 2021) for the neat data summary.
  • Hmisc (Harrel, 2020). To get correlation matrix with p-values.
  • Openxlsx (Schauwberber & Walker, 2020). To write dataframes into an Excel sheet.
